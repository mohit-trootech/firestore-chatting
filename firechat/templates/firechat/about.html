Firechat is a real-time group chat application built with Django and Firestore. It features a responsive UI using Tailwind CSS & Daisy UI, allowing users to communicate seamlessly while providing a smooth user experience.

Features
  • Realtime Chatting
    Instant messaging with friends and groups.
  • User Authentication
    Secure login and registration for users.
  • Profile Mangement
    Update your profile information easily.
  • User Profile View
    View the profiles of available users.
  • Online Status
    Show your online status to other users.
  • Login Stream
    Track your login streak to encourage daily usage.

Installation
Python 3.x
Django
Firebase Account
Firestore Database
Clone the repository
git clone https://github.com/mohit-trootech/firestore-chatting.git
cd firestore-chatting
Firestore Database
Create Virtual Environment
python -m venv venv
source venv/bin/activate
On Windows use `venv\Scripts\activate`
Install Required Packages
pip install -r requirements.txt
Create & Configure Firebase Account
Update Django Create Django Environment Files & Settings
Configure Database and Migrate Migrations
python manage.py makemigrations
python manage.py migrate
Start Development Server
python manage.py runserver
